FluidEvent::~FluidEvent

Destroy the Fluid Event:: Fluid Event object.

Note
Deletes any FluidEventFunction that refuses to unsubscribe. If this behavior did not occur, then any such subscribers would still be deleted per the std::map destructor (which calls the destructors for all non-pointer members). The intent here is to prevent EventFunctions from being unnecessarily deleted.
